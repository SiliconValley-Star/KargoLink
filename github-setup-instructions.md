# GitHub Repository Setup Instructions

## Step 1: Create GitHub Repository
1. Go to https://github.com
2. Click the "+" button in top right corner
3. Select "New repository"
4. Fill in repository details:
   - Repository name: `CargoLink`
   - Description: `Professional cargo tracking system with enterprise-grade architecture`
   - Set as Public or Private (your choice)
   - Do NOT initialize with README, .gitignore, or license
5. Click "Create repository"

## Step 2: Copy Repository URL
After creating the repository, copy the HTTPS URL shown (format: https://github.com/YOUR_USERNAME/CargoLink.git)

## Step 3: Commands to Run in Terminal
Once you have the URL, run these commands:

```bash
# Add remote origin (replace YOUR_REPO_URL with actual URL)
git remote add origin YOUR_REPO_URL

# Set main branch
git branch -M main

# Push to GitHub
git push -u origin main
```
